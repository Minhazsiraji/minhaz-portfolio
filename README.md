# Minhaz Siraji — Apparel Portfolio

A lightweight GitHub Pages portfolio for remote apparel sourcing and merchandising services.

## Before publishing

1. Open `index.html` and replace `YOUR-PROFESSIONAL-EMAIL@example.com`, `YOUR-LINKEDIN-URL`, and `YOUR-UPWORK-URL`.
2. Confirm that every experience statement is accurate. Add years or product categories only when you are comfortable publishing them.
3. Keep the demo workbook fictional and never upload employer, buyer, supplier, factory, price, or product data without written authorization.

## Publish free with GitHub Pages

1. Create a public GitHub repository, for example `minhaz-apparel-portfolio`.
2. Upload the contents of this folder to the repository root.
3. In the repository, open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**, choose `main` and `/ (root)`, then save.

GitHub will show the public portfolio URL after deployment.
